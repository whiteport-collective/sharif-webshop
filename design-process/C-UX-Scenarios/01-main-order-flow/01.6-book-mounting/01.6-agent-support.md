# 01.6 Agent Support — Book Mounting

**Companion to:** [01.6-Book Mounting](./01.6-book-mounting.md)
**Scenario:** 02 — Man Purchasing Flow, Path A (Assisted Purchase)
**Agent role at this step:** Scheduler Assistant

---

## What the Agent Does Here

After payment, the user needs to book a mounting appointment. The booking widget shows an accordion of days with available time slots. The agent can help the user pick a time, answer questions about the workshop, and confirm the booking.

This step only applies if the user selected a workshop delivery method. For home delivery, this step is skipped and the agent is not active.

---

## What the Agent Offers

When the booking step appears, the agent re-activates:

> *Agent:* Nå kan du bestille monteringstid. Vil du at jeg viser deg de nærmeste ledige tidene?

If the user accepts, the agent highlights (scrolls to and expands) the next available day in the accordion.

---

## Answering Questions About the Workshop

| Question | Agent response |
|----------|---------------|
| "Hvor er verkstedet?" | Gives address + "åpner i kart" link |
| "Når er det åpent?" | Workshop hours from knowledge base |
| "Hvor lang tid tar monteringen?" | "Vanligvis 30–60 minutter for fire dekk." |
| "Kan jeg vente der?" | Answers from workshop profile in knowledge base |
| "Kan jeg endre tid etterpå?" | "Ja — kontakt oss på chat eller telefon." (Path B handles this) |

---

## Booking Selection

The user must tap the time slot themselves — the agent does not book on their behalf.

The agent can narrow the choice:
> *Agent:* Er det noen dager som passer spesielt godt?

After the user taps a time slot:
> *Agent:* Flott — [dag] kl. [tid] på [verksted]. Vil du bekrefte?

The user taps **Bekreft montering** in the booking widget. The agent does not tap it.

---

## After Confirmation

> *Agent:* Perfekt! Du er booket inn. Du får en bekreftelse på e-post. Ha en fin tur! 🛞

The agent session ends. The order confirmation screen takes over.

---

## Agent Capabilities at This Step

| Capability | Allowed | Notes |
|-----------|---------|-------|
| Highlight next available day | ✓ | Scrolls to and expands first available day in accordion |
| Answer workshop questions | ✓ | Location, hours, duration |
| Suggest a time based on user preference | ✓ | "Morgon eller ettermiddag?" → narrows to matching slots |
| Select a time slot | ✗ | User taps — booking is a commitment |
| Submit booking confirmation | ✗ | User taps **Bekreft montering** |

---

## Reschedule via Path B

If the user needs to change their booking time after this step (different session, different day), they use Path B — Order Support — described in [Scenario 02, step 02.9](../../02-man-purchasing-flow/02-man-purchasing-flow.md#path-b--order-support-returning-customers).
